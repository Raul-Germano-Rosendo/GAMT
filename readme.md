# Estudos e Atividades - GAMT

Repositório utilizado para estudar, testar e organizar conteúdos relacionados às atividades realizadas durante o estágio no GAMT.

Aqui são armazenados:
- Anotações de estudo
- Testes e experimentos
- Scripts e códigos de apoio
- Materiais utilizados no aprendizado

O objetivo é centralizar o aprendizado e registrar o progresso durante o período de estágio para facilitar futuramente o desenvolvimento do relatório de estágio.
